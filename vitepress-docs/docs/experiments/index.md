# Experiments

Experiments 用于记录快速试验、局部原型和还没有进入长期结构的探索内容。

## 典型内容

- 新 Prompt 交互原型
- MCP Explorer 草图
- Workflow 可视化方案试验
- React Flow 节点布局实验
- 浏览器侧 AI 与存储机制试验

## 推荐节奏

1. 先用 feature 分支做最小原型。
2. 自动生成 `/preview/<branch-slug>/` 预览环境。
3. 方向稳定后再迁移到 `lab` 或 `playground`。

## 当前目标

这个目录先作为实验总入口，后续可以按主题继续拆分子目录。
