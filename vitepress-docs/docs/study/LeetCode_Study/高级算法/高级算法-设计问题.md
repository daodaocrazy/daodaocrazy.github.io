  4. 数据流的中位数

  + 语言：java

  + 思路：分别用最大堆和最小堆存储数据流的较小的一半和较大的一半。关键是如何调整两个堆。（调整不好会超时）

  + 代码（67ms，74.66%）：首先，保证大顶堆储量 >= 小顶堆储量（这样中位数要么是大顶堆第一个，要么相加*0.5）。每次添加后，对两个堆进行调整，如果大顶堆存在比小顶堆大的数，就需要和小顶堆调函数字（保证大顶堆存较小的一半）。我这个是先保证数量，再保证大小。
