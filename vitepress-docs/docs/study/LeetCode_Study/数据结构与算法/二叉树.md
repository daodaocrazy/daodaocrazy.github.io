### 二叉树的前序遍历

语言：java

思路：题目要求递归和迭代

代码1（0ms，100%）：递归

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode &#123;
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() &#123;&#125;
 *     TreeNode(int val) &#123; this.val = val; &#125;
 *     TreeNode(int val, TreeNode left, TreeNode right) &#123;
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     &#125;
 * &#125;

  public List&lt;Integer> preorderTraversal(TreeNode root) &#123;
    dfs(root);
    return res;
  &#125;

  public void dfs(TreeNode root)&#123;
    if(root==null)&#123;
      return;
    &#125;
    res.add(root.val);
    dfs(root.left);
    dfs(root.right);
  &#125;
&#125;
```

代码2（0ms，100%）：迭代。借用数据结构栈，暂存右节点。

前序：中左右，则暂存每次要遍历的节点（第一个root以及之后每次的右、左节点）。由于右子树最后遍历，所以压栈底的是右子树（如果有的话）。

```java
class Solution &#123;
  public List&lt;Integer> preorderTraversal(TreeNode root) &#123;

    List&lt;Integer> res = new LinkedList&lt;>();
    LinkedList&lt;TreeNode> stack = new LinkedList&lt;>();
    if(root!=null)&#123;
      stack.addFirst(root);
    &#125;
    while(!stack.isEmpty())&#123;
      TreeNode node = stack.pollFirst();
      res.add(node.val);
      if(node.right!=null)&#123;
        stack.addFirst(node.right);
      &#125;
      if(node.left!=null)&#123;
        stack.addFirst(node.left);
      &#125;
    &#125;
    return res;
  &#125;
&#125;
```

代码2（1ms，51.49%）：迭代。

这个需要注意的是后续遍历的右节点判断时，可能出现反复在当前节点和右节点之间访问的死循环，所以多用个临时节点判断上一个节点是否为右节点。

```java
class Solution &#123;
  public List&lt;Integer> postorderTraversal(TreeNode root) &#123;
    List&lt;Integer> res = new LinkedList&lt;>();
    LinkedList&lt;TreeNode> stack = new LinkedList&lt;>();
    TreeNode pre =null;
    while (root != null || !stack.isEmpty()) &#123;

      // 先一直往左
      while(root!=null)&#123;
        stack.addFirst(root);
        root = root.left;
      &#125;

      root = stack.pollFirst();
      if (root.right == null || root.right == pre) &#123;
        res.add(root.val);
        pre = root;
        root = null;
      &#125; else &#123;
        stack.addFirst(root);
        root = root.right;
      &#125;
      // 左右中
    &#125;
    return res;
  &#125;
&#125;
```

参考代码1（0ms）：

> [迭代解法，时间复杂度 O(n)，空间复杂度 O(n)](https://leetcode-cn.com/problems/binary-tree-postorder-traversal/solution/die-dai-jie-fa-shi-jian-fu-za-du-onkong-jian-fu-za/)

这个模版可以套用（用于前序、中序、后序）

这个思路很有意思：

原本前序遍历使用Stack暂存中间节点，List存遍历的结果值，这时候遍历顺序为“根左右”；那么我们要的后续遍历是“左右根”。

如果把Stack暂存节点变成暂存右节点，那么就"根左右"->"根右左"；

这时候再把List的添加顺序变成类似栈的加入到头部，那么最后输出的遍历顺序就完全颠倒，于是有"根左右"->"根右左"->"左右根"。

``` java
class Solution &#123;
  public List&lt;Integer> postorderTraversal(TreeNode root) &#123;
    LinkedList&lt;Integer> result = new LinkedList&lt;>();
    LinkedList&lt;TreeNode> stack = new LinkedList&lt;>();
    while (root != null || !stack.isEmpty()) &#123;
      if (root != null) &#123;
        stack.push(root);
        result.addFirst(root.val);
        root = root.right;
      &#125; else &#123;
        root = stack.pop();
        root = root.left;
      &#125;
    &#125;
    return result;
  &#125;
&#125;
```

### 二叉树的层序遍历

语言：java

思路：BFS。这类经典的BFS诀窍就是使用Queue队列

```java
    List&lt;List&lt;Integer>> res = new LinkedList&lt;>();
    Queue&lt;TreeNode> queue = new LinkedList&lt;>();

    if(root!=null)&#123;
      queue.add(root);
    &#125;

        &#125;
        if(cur.right!=null)&#123;
          queue.add(cur.right);
        &#125;
      &#125;

    &#125;

  &#125;

```
