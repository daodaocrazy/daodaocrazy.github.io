## 02 二叉搜索树简介

### 验证二叉搜索树

语言：java

思路：递归左右子树，如果出现非 左&lt;根&lt;右的情况，则返回false。这里需要注意的是左右字数的递归调用，递归右子树时，主要考虑原节点是否大于等于右子树的值；递归左子树时主要判断原节点是小于等于原节点。

代码（0ms）：

```java
class Solution &#123;
  public boolean isValidBST(TreeNode root) &#123;
    return dfs(root, null, null);
  &#125;

  public boolean dfs(TreeNode cur, Integer low, Integer high) &#123;
    if (cur == null) &#123;
      return true;
    &#125;
    if (low != null && cur.val &lt;= low) &#123;
      return false;
    &#125;
    if (high != null && cur.val >= high) &#123;
      return false;
    &#125;
    if (!dfs(cur.right, cur.val, high)) &#123;
      return false;
    &#125;
    if (!dfs(cur.left, low, cur.val)) &#123;
      return false;
    &#125;
    return true;
  &#125;
&#125;
```

### 二叉搜索树迭代器

语言：java

思路：就中序遍历，用`List`存储遍历到的值。但是这个不符合要求的空间复杂度`O(h)`

代码（23ms，92.96%）：

```java
class BSTIterator &#123;
  List&lt;Integer> res;
  int index = 0;
  public BSTIterator(TreeNode root) &#123;
    res = new ArrayList&lt;>();
    dfs(root);
  &#125;

  public void dfs(TreeNode root)&#123;
    if(root==null)&#123;
      return;
    &#125;
    dfs(root.left);
    res.add(root.val);
    dfs(root.right);
  &#125;

  /** @return the next smallest number */
  public int next() &#123;
    return res.get(index++);
  &#125;

  /** @return whether we have a next smallest number */
  public boolean hasNext() &#123;
    return index&lt;res.size();
  &#125;
&#125;
```

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode &#123;
 * int val;
 * TreeNode left;
 * TreeNode right;
 * TreeNode(int x) &#123; val = x; &#125;
 * &#125;

class BSTIterator &#123;

  Stack&lt;TreeNode> stack;

  public BSTIterator(TreeNode root) &#123;

    // Stack for the recursion simulation
    this.stack = new Stack&lt;TreeNode>();

    // Remember that the algorithm starts with a call to the helper function
    // with the root node as the input
    this._leftmostInorder(root);
  &#125;

  private void _leftmostInorder(TreeNode root) &#123;

    // For a given node, add all the elements in the leftmost branch of the tree
    // under it to the stack.
    while (root != null) &#123;
      this.stack.push(root);
      root = root.left;
    &#125;
  &#125;

  /**
     * @return the next smallest number
     */
  public int next() &#123;
    // Node at the top of the stack is the next smallest element
    TreeNode topmostNode = this.stack.pop();

    // Need to maintain the invariant. If the node has a right child, call the 
    // helper function for the right child
    if (topmostNode.right != null) &#123;
      this._leftmostInorder(topmostNode.right);
    &#125;

    return topmostNode.val;
  &#125;

  /**
     * @return whether we have a next smallest number
     */
  public boolean hasNext() &#123;
    return this.stack.size() > 0;
  &#125;
&#125;
```
