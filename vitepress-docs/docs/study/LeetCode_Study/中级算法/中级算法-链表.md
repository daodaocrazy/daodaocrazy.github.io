  2. 奇偶链表

  + 语言：java

  + 思路：一路奇数，一路偶数

  + 代码（0ms,41.3MB）：

    ```java

                    tmp2 = tmp2.next;
                &#125;

                return head;
    ```

  3. 相交链表
  
  + 语言：java
  
  + 思路：相交后，走的长度肯定一样，那么如果从尾向头看，就是只有长度相同的地方才是可能相交的地方。反正时间要求是O(n)，可以先获取上下长度，然后到长度相同的位置开始判断。
  
  + 代码（1ms，43.1MB，100%）：
  
    ```java
    public class Solution &#123;
        public ListNode getIntersectionNode(ListNode headA, ListNode headB) &#123;

                return null;
            int len1=0,len2=0;
            ListNode hA=headA,hB=headB;
            while(!(hA==null&&hB==null))&#123;
                if(hA!=null)&#123;
                    ++len1;
                    hA=hA.next;
                &#125;
                if(hB!=null)&#123;
                    ++len2;
                    hB=hB.next;
                &#125;
            &#125;
            hA = headA;
            hB = headB;
            while(len1>len2)&#123;
                hA = hA.next;
                --len1;
            &#125;
            while(len2>len1)&#123;
                hB = hB.next;
                --len2;
            &#125;

                hA = hA.next;
                hB = hB.next;
            &#125;
            return hA;
        &#125;
    &#125;
    ```
  
  + 参考代码（1ms，超精简写法）：他这个很巧妙，如果1和2相交，那么迟早会碰头（因为走的步数是一样的，碰头的只可能是相交点）。如果1和2不相交，那么走对方的路线，也迟早因为走的步数都是A+B而走向null。
  
    ```java

            while(l1!=l2)&#123;
                l1=(l1==null)?headB:l1.next;
                l2=(l2==null)?headA:l2.next;
            &#125;
            return l1;
        &#125;
    &#125;
    ```
