# 🧭 技术地图

> 🗺️ 从技术主题到内容分布，查看当前关注版图

## 🎯 核心技能

### 🏗️ 后端开发

- **Java** - 主语言，深入 JVM、并发、设计模式
- **Go** - 现代化系统编程语言
- **Spring** - 企业级应用开发框架
- **Netty** - 高性能网络编程框架

### 🐘 大数据

- **Hadoop** - 分布式基础架构
- **Spark** - 大规模数据处理引擎
- **Flink** - 流批一体计算框架
- **Kafka** - 分布式消息队列
- **ClickHouse** - 列式数据库
- **HBase** - NoSQL 数据库

### 🗄️ 数据存储

- **MySQL** - 关系型数据库
- **Redis** - 内存数据库/缓存
- **MongoDB** - 文档数据库
- **Elasticsearch** - 搜索引擎

### ⚛️ 前端开发

- **Vue** - 渐进式 JavaScript 框架
- **React** - UI 构建库
- **JavaScript** - 网页交互

### 🐳 DevOps

- **Docker** - 容器化技术
- **Kubernetes** - 容器编排平台
- **Jenkins** - 持续集成工具
- **CICD** - 持续集成与部署

## 🌟 学习路线

1. **夯实基础**: 数据结构、算法、计算机网络、操作系统
2. **精通一门**: 从 Java 入手，深入学习核心原理
3. **拓展生态**: 学习 Spring 全家桶，构建 Web 应用
4. **拥抱分布式**: 引入大数据、缓存、消息队列
5. **深入底层**: JVM 调优、性能优化、架构设计
6. **持续实践**: LeetCode 刷题、项目实战、源码阅读

- 🎯 **推荐系统** - 个性化推荐算法

- 🌐 **网络安全** - Web 安全、加密解密
- 🎨 **设计模式** - 代码设计原则
